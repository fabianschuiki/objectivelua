Detailed Description

All the objects that are likely to be exposed to Lua should inherit from this superclass. This fasciliates function exposure and handling of the instance pairs between C++ and Lua.

Creates a new LuaExposable instance and exposes it to the Lua state passed. The given class name is used to lookup the Lua metatable that contains the class inteface. If leaveOnStack is true, the resulting object is left on the Lua stack so it may be used later on.
